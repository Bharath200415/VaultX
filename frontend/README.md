# VaultX Frontend

This is the React frontend for **VaultX**, a mini cloud storage app.

For full project initialization, environment setup, and detailed instructions for both **Linux** and **Windows**, please refer to the main documentation:

👉 **[View Main README.md](../README.md)**

---

## 🔥 Frontend Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Access the application:
   `http://localhost:5173`
